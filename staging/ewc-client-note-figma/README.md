Elliott Wave Club client note template

Files:
- `page-1-summary.svg`
- `page-2-yearly-chart.svg`
- `page-3-weekly-chart.svg`

How to use in Figma:
1. Open a new Figma file.
2. Drag all three SVG files onto the canvas.
3. Each file will import as its own editable frame.
4. Replace the monogram mark with the final Elliott Wave Club logo if needed.
5. Replace the chart placeholders on pages 2 and 3 with chart images using image fill or by placing the images inside the chart frame.
6. Update the title, copy, levels, and captions per client note.
7. Export all three frames as a single PDF from Figma.

Template notes:
- Canvas size is `1600 x 1236` px per page, matching US Letter landscape ratio.
- The style is tuned for a dark premium financial note with chart-first page structure.
- Shared header and footer regions are visually consistent across all three pages so they can be turned into Figma components after import.
